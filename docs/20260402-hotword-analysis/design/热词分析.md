# 热词分析模块

## 一、模块概述

热词分析模块是对现有热词中心的扩展，新增热词类型字段和热词分析任务功能。

### 变更内容

| 变更项 | 说明 |
|--------|------|
| hot_word 表 | 新增 type 字段 |
| hot_word_task 表 | 新增 model 字段 |
| QConfig 配置 | 新增热词类型配置 |
| 热词管理页面 | 新增类型筛选 |
| 热词任务页面 | 新增「热词分析」Tab |

---

## 二、数据库变更

### 2.1 hot_word 表新增字段

```sql
ALTER TABLE `hot_word` ADD COLUMN `type` VARCHAR(64) DEFAULT NULL COMMENT '热词类型（从QConfig获取）' AFTER `tags`;
ALTER TABLE `hot_word` ADD KEY `idx_type` (`type`);
```

**type 字段说明：**
- 值来源于 QConfig 配置的 key，如 `poiAnalysis`、`platAnalysis`
- 具体的 `type` 和 `subType` 在 QConfig 中维护映射关系

### 2.2 hot_word_task 表新增字段

```sql
ALTER TABLE `hot_word_task` ADD COLUMN `model` VARCHAR(32) NOT NULL DEFAULT 'default' COMMENT '模型标识：default/deepseek/qianwen等' AFTER `type`;
```

**model 字段说明：**

| 值 | 含义 |
|----|------|
| default | 默认模型 |
| deepseek | DeepSeek 模型 |
| qianwen | 通义千问模型 |
| doubao | 豆包模型 |

## 三、QConfig 配置

### 3.1 热词类型配置

**配置 Key：** `hotword.type.config`

**配置内容：**

```json
{
    "poiAnalysis": {
        "type": "analysis_query",
        "subType": "poi",
        "name": "POI分析查询",
        "description": "针对POI相关的分析查询热词"
    },
    "platAnalysis": {
        "type": "analysis_query",
        "subType": "platform",
        "name": "平台分析查询",
        "description": "针对平台层面的分析查询热词"
    }
}
```

**字段说明：**

| 字段 | 说明 |
|------|------|
| key (poiAnalysis) | 热词类型标识，存储在 hot_word.type 字段 |
| type | 业务类型：analysis_query 等 |
| subType | 业务子类型：poi/platform 等 |
| name | 显示名称 |
| description | 描述说明 |

## 四、接口变更

### 4.1 热词接口变更

**GET /api/hotWord/list**

新增请求参数：

| 参数 | 类型 | 说明 |
|------|------|------|
| type | String | 热词类型筛选 |

响应新增字段：

| 字段 | 类型 | 说明 |
|------|------|------|
| type | String | 热词类型 |
| typeName | String | 热词类型名称（从配置获取） |

**POST /api/hotWord/add**

请求参数新增：

| 参数 | 类型 | 说明 |
|------|------|------|
| type | String | 热词类型 |

### 4.2 热词任务接口变更

**GET /api/hotWordTask/list**

请求参数不变，type 支持 `analysis` 值。

**POST /api/hotWordTask/add**

请求参数新增：

| 参数 | 类型 | 说明 |
|------|------|------|
| model | String | 模型标识，默认 default |

**热词分析任务 params 示例：**

```json
{
    "type": "poiAnalysis",
    "count": 10
}
```

**热词分析任务 result 示例：**

```json
{
    "total": 10,
    "type": "poiAnalysis",
    "words": [
        {"word": "北京天安门门票", "type": "poiAnalysis"},
        {"word": "故宫开放时间", "type": "poiAnalysis"}
    ]
}
```
