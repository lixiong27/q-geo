# 数据中心

## 一、模块概述

数据中心负责展示各模块的核心统计指标，通过实时聚合查询各业务表数据，为运营决策提供数据支持。

### 功能说明

- 展示热词、扩词、发布、渠道等核心指标
- 支持按时间范围筛选（今天、近7天、近30天、自定义）
- 展示热词来源分布、发布渠道分布等图表
- 数据实时计算，无需定时任务

---

## 二、数据来源

数据中心不创建独立的数据库表，所有数据通过聚合查询各业务表实时计算：

| 统计项 | 数据来源表 | 计算方式 |
|--------|-----------|----------|
| 热词数量 | `hot_word` | COUNT + 时间筛选 |
| 扩词总数 | `hot_word_task` (type=expand) | SUM(result.questions.length) |
| 发布文章数 | `content` (status=2) | COUNT |
| 活跃渠道 | `publish_channel` (status=active) | COUNT |
| 热词来源分布 | `hot_word` | GROUP BY source_type |
| 发布渠道分布 | `publish_task` | GROUP BY channel_id |

## 三、接口设计

### 3.1 获取数据中心全部数据

**接口：** `GET /api/dataCenter/all`

**请求参数：**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| timeRange | String | 否 | 时间范围：today/sevenDays/thirtyDays/custom，默认 sevenDays |
| startDate | String | 否 | 自定义开始日期，timeRange=custom 时必填 |
| endDate | String | 否 | 自定义结束日期，timeRange=custom 时必填 |
| trendDays | Integer | 否 | 趋势图天数，默认 7 |

**响应参数：**

| 参数 | 类型 | 说明 |
|------|------|------|
| overview | Object | 统计概览 |
| overview.hotWordCount | Integer | 热词数量 |
| overview.hotWordTodayNew | Integer | 今日新增热词 |
| overview.expandCount | Integer | 扩词总数 |
| overview.expandTodayNew | Integer | 今日扩词数 |
| overview.publishCount | Integer | 发布文章数 |
| overview.publishMonthNew | Integer | 本月新增发布数 |
| overview.activeChannelCount | Integer | 活跃渠道数 |
| overview.activeChannelChange | String | 渠道变化描述 |
| hotWordSourceDistribution | Array | 热词来源分布 |
| hotWordSourceDistribution[].source | String | 来源类型：weibo/zhihu/baidu/manual |
| hotWordSourceDistribution[].sourceName | String | 来源名称 |
| hotWordSourceDistribution[].count | Integer | 数量 |
| hotWordSourceDistribution[].percentage | Decimal | 占比(%) |
| publishChannelDistribution | Array | 发布渠道分布 |
| publishChannelDistribution[].channelId | Long | 渠道ID |
| publishChannelDistribution[].channelName | String | 渠道名称 |
| publishChannelDistribution[].count | Integer | 发布数量 |
| publishChannelDistribution[].percentage | Decimal | 占比(%) |
| dailyTrend | Object | 每日趋势 |
| dailyTrend.hotWord | Array | 热词趋势 |
| dailyTrend.expand | Array | 扩词趋势 |
| dailyTrend.publish | Array | 发布趋势 |
| dailyTrend.*[].date | String | 日期 |
| dailyTrend.*[].dayOfWeek | String | 星期 |
| dailyTrend.*[].count | Integer | 数量 |

## 四、数据流向

```
┌─────────────────────────────────────────────────────────────┐
│                    数据中心数据流                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   ┌──────────────┐  ┌──────────────┐  ┌──────────────┐     │
│   │ hot_word     │  │ content      │  │publish_channel│    │
│   │ hot_word_task│  │ publish_task │  │              │     │
│   └──────┬───────┘  └──────┬───────┘  └──────┬───────┘     │
│          │                 │                 │              │
│          └────────────┬────┴────────────────┘              │
│                       │                                     │
│                       ▼                                     │
│          ┌─────────────────────────┐                       │
│          │   实时聚合查询           │                       │
│          │   (COUNT/SUM/GROUP BY)  │                       │
│          └────────────┬────────────┘                       │
│                       │                                     │
│                       ▼                                     │
│          ┌─────────────────────────┐                       │
│          │   数据中心接口返回       │                       │
│          │   (无持久化存储)         │                       │
│          └─────────────────────────┘                       │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**说明：**
- 数据中心不存储数据，所有指标通过实时查询计算
- 可根据查询频率考虑增加缓存层（Redis）
- 复杂聚合查询可考虑异步加载
