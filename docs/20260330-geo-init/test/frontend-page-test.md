# 前端页面测试用例

## 测试概述

本文档包含 GEO 运营平台所有前端页面的测试用例，按页面模块划分。

**测试环境要求：**
- 前端服务启动正常 (http://localhost:3000)
- 后端服务可访问
- 浏览器：Chrome/Firefox 最新版

**页面路由：**
| 页面 | 路由 | 说明 |
|------|------|------|
| 热词中心 | /hotWord | 默认首页 |
| 内容中心 | /content | 内容管理 |
| GEO分析 | /geoMonitor | 监控面板 |
| 数据中心 | /dataCenter | 数据统计 |
| 发布中心 | /publish | 发布管理 |
