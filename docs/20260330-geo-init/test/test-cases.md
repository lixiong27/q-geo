# GEO 运营平台测试用例

## 测试概述

本文档按模块组织测试用例，包含后端接口和前端页面测试。

**测试环境：**
- 后端：http://localhost:8080
- 前端：http://localhost:3000
- 数据库：mkt_ares_live_beta

**通用响应格式：**
```json
{
    "code": 0,
    "message": "success",
    "data": {}
}
```
