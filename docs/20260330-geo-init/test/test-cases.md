# GEO 运营平台测试用例

## 测试概述

本文档按模块组织测试用例，包含后端接口和前端页面测试。

**测试环境：**
- 后端：http://localhost:8080
- 前端：http://localhost:3000
- 数据库：mkt_ares_live_beta

**通用响应格式：**
```json
{
    "code": 0,
    "message": "success",
    "data": {}
}
```

## 一、热词中心模块

### 1.1 热词管理

#### 后端接口

| 接口 | 方法 | 路径 |
|------|------|------|
| 查询列表 | GET | /api/hotWord/list |
| 新增 | POST | /api/hotWord/add |
| 批量导入 | POST | /api/hotWord/import |
| 更新 | POST | /api/hotWord/update |
| 删除 | POST | /api/hotWord/delete |

#### 接口测试用例

| 编号 | 用例 | 请求 | 预期结果 |
|------|------|------|----------|
| HW-API-001 | 查询全部 | GET /api/hotWord/list | 返回分页列表 |
| HW-API-002 | 关键词搜索 | keyword="机票" | 返回匹配热词 |
| HW-API-003 | 来源筛选 | sourceType=0 | 返回手动添加热词 |
| HW-API-004 | 新增热词 | POST {word, tags} | code=0, 返回ID |
| HW-API-005 | 重复新增 | 相同word | code=非0 |
| HW-API-006 | 批量导入 | POST {words: [...]} | 返回成功数量 |
| HW-API-007 | 更新热词 | POST {id, word, tags} | code=0 |
| HW-API-008 | 删除热词 | POST {id} | code=0 |
