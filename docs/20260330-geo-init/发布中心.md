# 发布中心

## 一、模块概述

发布中心负责渠道管理和内容发布功能。

### Tab 结构

| Tab | 功能 |
|-----|------|
| 发布队列 | 发布任务展示、新建发布任务、查看发布状态 |
| 渠道管理 | 渠道展示、启用/禁用、配置编辑 |

---

## 二、数据表设计

### 2.1 发布渠道表 (publish_channel)

```sql
CREATE TABLE `publish_channel` (
    `id`            BIGINT UNSIGNED  NOT NULL AUTO_INCREMENT COMMENT '主键ID',
    `name`          VARCHAR(64)      NOT NULL DEFAULT '' COMMENT '渠道名称',
    `code`          VARCHAR(32)      NOT NULL DEFAULT '' COMMENT '渠道编码',
    `icon`          VARCHAR(64)      DEFAULT NULL COMMENT '图标（emoji或图片路径）',
    `config`        TEXT             DEFAULT NULL COMMENT '配置信息（JSON格式）',
    `status`        TINYINT(1)       NOT NULL DEFAULT 1 COMMENT '状态：0-禁用 1-启用',
    `is_builtin`    TINYINT(1)       NOT NULL DEFAULT 0 COMMENT '是否内置：0-否 1-是',
    `create_time`   DATETIME         NOT NULL DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
    `update_time`   DATETIME         NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '修改时间',
    PRIMARY KEY (`id`),
    UNIQUE KEY `uk_code` (`code`),
    KEY `idx_status` (`status`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='发布渠道表';
```

**status 说明：**

| 值 | 含义 |
|----|------|
| 0 | 禁用 |
| 1 | 启用 |

**is_builtin 说明：**

| 值 | 含义 |
|----|------|
| 0 | 自定义渠道 |
| 1 | 系统内置渠道 |

**预置渠道数据：**

```sql
INSERT INTO `publish_channel` (`name`, `code`, `icon`, `status`, `is_builtin`) VALUES
('微博', 'weibo', '📱', 1, 1),
('知乎', 'zhihu', '📝', 1, 1),
('微信公众号', 'wechat', '💬', 1, 1),
('小红书', 'xiaohongshu', '📕', 1, 1),
('抖音', 'douyin', '🎵', 1, 1),
('B站', 'bilibili', '📺', 1, 1);
```

**config 示例：**

```json
{
    "accountId": "公众号ID",
    "webhookUrl": "https://webhook.example.com/publish"
}
```

### 2.2 发布任务表 (publish_task)

```sql
CREATE TABLE `publish_task` (
    `id`            BIGINT UNSIGNED  NOT NULL AUTO_INCREMENT COMMENT '主键ID',
    `name`          VARCHAR(128)     NOT NULL DEFAULT '' COMMENT '任务名称',
    `content_id`    BIGINT UNSIGNED  NOT NULL COMMENT '关联内容ID',
    `channel_ids`   TEXT             DEFAULT NULL COMMENT '发布渠道ID列表（JSON数组）',
    `publish_time`  DATETIME         DEFAULT NULL COMMENT '定时发布时间',
    `status`        TINYINT(1)       NOT NULL DEFAULT 0 COMMENT '状态：0-待发布 1-发布中 2-已发布 3-失败',
    `result`        TEXT             DEFAULT NULL COMMENT '发布结果（JSON格式）',
    `created_by`    VARCHAR(64)      NOT NULL DEFAULT '' COMMENT '创建人',
    `create_time`   DATETIME         NOT NULL DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
    `update_time`   DATETIME         NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '修改时间',
    `completed_at`  DATETIME         DEFAULT NULL COMMENT '完成时间',
    PRIMARY KEY (`id`),
    KEY `idx_content_id` (`content_id`),
    KEY `idx_status` (`status`),
    KEY `idx_publish_time` (`publish_time`),
    KEY `idx_create_time` (`create_time`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='发布任务表';
```

**status 说明：**

| 值 | 含义 |
|----|------|
| 0 | 待发布 |
| 1 | 发布中 |
| 2 | 已发布 |
| 3 | 失败 |

**channel_ids 示例：**

```json
[1, 2, 3]
```

**result 示例：**

```json
{
    "total": 3,
    "items": [
        {
            "channelId": 1,
            "channelName": "微博",
            "status": "success",
            "publishedAt": "2026-03-31 14:30:00",
            "externalId": "weibo_123456"
        },
        {
            "channelId": 2,
            "channelName": "知乎",
            "status": "failed",
            "error": "认证失败"
        }
    ]
}
```

## 五、接口设计

### 5.1 发布任务接口

| 接口 | 方法 | 参数示例 |
|------|------|---------|
| `/api/publishTask/list` | GET | `?status=0&page=1&size=20` |
| `/api/publishTask/add` | POST | `{"name":"任务名","contentId":1,"channelIds":[1,2],"publishTime":"2026-03-31 18:00:00"}` |
| `/api/publishTask/detail` | GET | `?id=1` |
| `/api/publishTask/cancel` | POST | `{"id":1}` |
| `/api/publishTask/retry` | POST | `{"id":1}` |

### 5.2 发布渠道接口

| 接口 | 方法 | 参数示例 |
|------|------|---------|
| `/api/publishChannel/list` | GET | `?status=1` |
| `/api/publishChannel/detail` | GET | `?id=1` |
| `/api/publishChannel/update` | POST | `{"id":1,"config":{...}}` |
| `/api/publishChannel/toggleStatus` | POST | `{"id":1,"status":0}` |

### 5.3 内容选择接口

| 接口 | 方法 | 参数示例 |
|------|------|---------|
| `/api/content/publishable` | GET | `?keyword=天安门&page=1&size=10` |

## 六、模块关联

```
┌─────────────┐     content_id      ┌─────────────┐
│  内容中心    │ ──────────────────►│  发布中心    │
│  content    │                     │ publish_task│
│  表         │                     │             │
└─────────────┘                     └──────┬──────┘
                                           │
                                           │ channel_ids
                                           ▼
                                    ┌─────────────┐
                                    │ publish_    │
                                    │ channel     │
                                    └─────────────┘
```

**关联说明：**
- 发布任务通过 `content_id` 关联内容中心的 `content` 表
- 发布任务通过 `channel_ids` 关联 `publish_channel` 表
- 内容中心只提供 `status=2`（已发布状态）的内容供选择
