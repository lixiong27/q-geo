# 内容中心

## 一、模块概述

内容中心负责内容的管理和生成功能。

### Tab 结构

| Tab | 功能 |
|-----|------|
| 内容管理 | 内容展示、状态筛选、批量操作 |
| 内容生成任务 | 新建生成任务、任务列表、结果查看 |
| 手动生成内容 | 手动编辑器创建内容 |

---

## 二、数据表设计

### 2.1 内容表 (content)

```sql
CREATE TABLE `content` (
    `id`               BIGINT UNSIGNED  NOT NULL AUTO_INCREMENT COMMENT '主键ID',
    `title`            VARCHAR(256)     NOT NULL DEFAULT '' COMMENT '标题',
    `body`             MEDIUMTEXT       DEFAULT NULL COMMENT '正文内容',

    -- 来源
    `source_type`      TINYINT(1)       NOT NULL DEFAULT 0 COMMENT '来源类型：0-手动创建 1-任务生成',
    `source_task_id`   BIGINT UNSIGNED  NOT NULL DEFAULT 0 COMMENT '来源任务ID（source_type=1时有效）',
    `generate_method`  VARCHAR(32)      DEFAULT NULL COMMENT '生成方式：llm-LLM生成 claw-Claw抓取（source_type=1时有效）',

    -- 附件
    `attachments`      TEXT             DEFAULT NULL COMMENT '附件（JSON数组）',

    -- 统计
    `word_count`       INT UNSIGNED     NOT NULL DEFAULT 0 COMMENT '字数',

    -- 状态
    `status`           TINYINT(1)       NOT NULL DEFAULT 0 COMMENT '状态：0-草稿 1-待审核 2-已发布',

    -- 审计
    `created_by`       VARCHAR(64)      NOT NULL DEFAULT '' COMMENT '创建人',
    `create_time`      DATETIME         NOT NULL DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
    `update_time`      DATETIME         NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '修改时间',

    PRIMARY KEY (`id`),
    KEY `idx_title` (`title`),
    KEY `idx_source_type` (`source_type`),
    KEY `idx_source_task_id` (`source_task_id`),
    KEY `idx_status` (`status`),
    KEY `idx_create_time` (`create_time`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='内容表';
```

**source_type 说明：**

| 值 | 含义 | 说明 |
|----|------|------|
| 0 | 手动创建 | 手动生成内容 Tab 创建 |
| 1 | 任务生成 | 内容生成任务产生 |

**generate_method 说明：**

| 值 | 含义 |
|----|------|
| llm | LLM 生成 |
| claw | Claw 抓取 |

**status 说明：**

| 值 | 含义 |
|----|------|
| 0 | 草稿 |
| 1 | 待审核 |
| 2 | 已发布 |

**attachments 示例：**

```json
[
    {"type": "image", "url": "https://...", "name": "封面图.png"},
    {"type": "video", "url": "https://...", "name": "演示视频.mp4"}
]
```

### 2.2 内容生成任务表 (content_task)

```sql
CREATE TABLE `content_task` (
    `id`               BIGINT UNSIGNED  NOT NULL AUTO_INCREMENT COMMENT '主键ID',
    `name`             VARCHAR(128)     NOT NULL DEFAULT '' COMMENT '任务名称',

    -- 生成方式
    `generate_method`  VARCHAR(32)      NOT NULL DEFAULT '' COMMENT '生成方式：llm-LLM生成 claw-Claw抓取',

    -- 输入数据
    `input_data`       TEXT             DEFAULT NULL COMMENT '输入数据（JSON格式，存标题列表）',

    -- 模板配置
    `template_code`    VARCHAR(64)      DEFAULT NULL COMMENT '模板编码（对应配置文件中的模板）',

    -- 状态与结果
    `status`           TINYINT(1)       NOT NULL DEFAULT 0 COMMENT '状态：0-待执行 1-执行中 2-已完成 3-失败',
    `result`           TEXT             DEFAULT NULL COMMENT '执行结果（JSON格式）',

    -- 审计
    `created_by`       VARCHAR(64)      NOT NULL DEFAULT '' COMMENT '创建人',
    `create_time`      DATETIME         NOT NULL DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
    `update_time`      DATETIME         NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '修改时间',
    `completed_at`     DATETIME         DEFAULT NULL COMMENT '完成时间',

    PRIMARY KEY (`id`),
    KEY `idx_generate_method` (`generate_method`),
    KEY `idx_status` (`status`),
    KEY `idx_create_time` (`create_time`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='内容生成任务表';
```

**status 说明：**

| 值 | 含义 |
|----|------|
| 0 | 待执行 |
| 1 | 执行中 |
| 2 | 已完成 |
| 3 | 失败 |

**input_data 示例：**

```json
{
    "title": "文章标题"
}
```

**result 示例：**

```json
{
    "contentId": 101
}
```

## 五、接口设计

### 5.1 内容接口

| 接口 | 方法 | 参数示例 |
|------|------|---------|
| `/api/content/list` | GET | `?sourceType=0&status=0&page=1&size=20` |
| `/api/content/add` | POST | `{"title":"标题","body":"正文"}` |
| `/api/content/detail` | GET | `?id=1` |
| `/api/content/update` | POST | `{"id":1,"title":"标题","body":"正文"}` |
| `/api/content/delete` | POST | `{"id":1}` |

### 5.2 内容生成任务接口

| 接口 | 方法 | 参数示例 |
|------|------|---------|
| `/api/contentTask/list` | GET | `?status=0&page=1&size=20` |
| `/api/contentTask/add` | POST | `{"name":"任务名","generateMethod":"llm","inputData":{"title":"文章标题"},"templateCode":"short_article"}` |
| `/api/contentTask/detail` | GET | `?id=1` |
| `/api/contentTask/cancel` | POST | `{"id":1}` |
| `/api/contentTask/retry` | POST | `{"id":1}` |
