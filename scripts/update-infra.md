# 工程上下文更新说明

## 更新命令

当工程能力发生变化时，使用以下命令触发更新：

```
/update-infra
```

## 更新范围

### 业务模块 (infra/biz/)

新增模块时：
1. 在 `infra/biz/` 下创建 `模块名.md`
2. 在 `infra.md` 中添加导航条目

更新模块时：
- 新增 API → 更新 API 清单
- 新增 Mapper → 更新 Mapper 列表
- 新增前端组件 → 更新前端组件列表
- 新增下游调用 → 更新下游调用列表
- 新增 QConfig → 更新相关配置列表
- 新增执行器 → 更新任务执行器列表

### 技术组件 (infra/tec/)

新增技术组件时：
- 在 `infra/tec/components.md` 中添加组件说明

新增代码规范时：
- 在 `infra/tec/coding-style.md` 中补充规范

## 更新检查清单

- [ ] API 清单是否完整
- [ ] Mapper 是否遗漏
- [ ] 前端组件是否对应
- [ ] 下游调用是否记录
- [ ] QConfig 配置是否更新
- [ ] 执行器是否添加
